# 실전 문제
시스템 테스팅의 근간은 상위 레벨의 basis이다.
- 리스크 분석서
- 요구사항 명세(프로그램 명세와의 차이점은? 프로그래밍의 유지 보수가 가능하도록 프로그램의 구조와 기능을)
- 비즈니스 프로세스
- 유즈케이스
- 기타 비즈니스 레벨의 시스템 동작 명세
- os 및 시스템 리소스와의 상화작용 명세

보안성 테스팅은 기능성 테스팅의 한 부분으로 보는 것이 일반적이다.
