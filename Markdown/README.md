# 링크 상대 경로
```
[보이는 글](링크)
```
링크에 상대 경로 입력 가능. ex) ./ or ./README.md

## 경로 상의 띄어쓰기 처리
경로 상에 또는 문서에 띄어쓰기가 포함된 경우 percent encoding rule에 따라 표기된다.[^stackoverflow][^wikipedia_percent_encoding_rule]
